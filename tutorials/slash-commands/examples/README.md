# Examples

This folder contains example projects that demonstrate the slash commands workflow.

## RAG Chatbot

A complete example showing how to use `/design`, `/plan`, and `/task` to build a retrieval-augmented generation chatbot.

**Tech Stack:**
- FastAPI + Python
- PostgreSQL with pgvector
- OpenAI embeddings
- Claude for responses
- Tailwind CSS frontend

See [rag-chatbot/README.md](rag-chatbot/README.md) for setup instructions.

## Workflow Artifacts

The `workspace/` folder contains the design document and task files generated by the slash commands:

- `workspace/designs/rag-chatbot.md` - Technical design document
- `workspace/tasks/rag-chatbot/` - Individual task files

These artifacts demonstrate the complete workflow from design to implementation.
