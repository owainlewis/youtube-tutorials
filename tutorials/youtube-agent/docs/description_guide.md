# YouTube Video Description Guide

Best practices for writing YouTube video descriptions.

## Structure

```
[First 150 chars - Hook/SEO]

[Main description - 2-3 paragraphs]

[Timestamps]

[Resources/Links]

[Social links/CTA]
```

## First 150 Characters

The first ~150 characters appear in search results. Make them count.

**Good:**
> Learn how to build AI agents that work with any coding assistant. No platform lock-in, just portable scripts.

**Bad:**
> Hey everyone, in this video I'm going to show you something really cool that I've been working on...

## Timestamps

Always include timestamps for videos over 3 minutes:

```
0:00 Intro
1:30 What is a Micro-Agent?
3:30 The AGENTS.md Standard
6:00 Building the Tool
10:00 Authentication
11:30 Why This Works
14:00 Summary
```

## Links Section

Include relevant resources:

```
Resources:
- GitHub Repo: https://github.com/...
- Documentation: https://...
- Related video: https://youtube.com/watch?v=...
```

## Tags

Include 5-15 relevant tags:
- Primary topic (e.g., "AI agents")
- Related technologies (e.g., "Claude", "Python")
- Format/type (e.g., "tutorial", "coding")

## Privacy Settings

| Setting | When to Use |
|---------|-------------|
| private | During editing/review |
| unlisted | For early feedback |
| public | Final release |

**Always upload as unlisted first, review, then make public.**
